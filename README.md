# 🚢 Titanic AI Dashboard

## 📌 Project Overview

The Titanic AI Dashboard is an interactive data analytics application built using Streamlit, Pandas, and Plotly. It provides insights into the Titanic passenger dataset through data cleaning, KPI metrics, filters, and interactive visualizations.

---

## 🎯 Features

### 📊 Dataset Overview

* Displays dataset shape
* Shows first 5 records of the dataset

### 🧹 Data Cleaning

* Handles missing values in Age and Embarked columns
* Removes Cabin column
* Displays missing values before and after cleaning

### 📈 Key Performance Indicators (KPIs)

* Total Passengers
* Total Survivors
* Average Age
* Survival Rate Percentage

### 🎛️ Interactive Filters

* Filter by Gender
* Filter by Passenger Class

### 📉 Visualizations

1. Survival Distribution (Bar Chart)
2. Gender Distribution (Pie Chart)
3. Age Distribution (Histogram)
4. Survival Rate by Passenger Class (Bar Chart)
5. Fare Distribution (Box Plot)

### ❄️ UI Enhancements

* Custom dashboard header
* Sidebar styling
* Snow animation effect
* Interactive Plotly charts
